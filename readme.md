#  da Página
**Como será a Tecnologia no Futuro** (Focado no ecossistema de Inovação Digital, Agronegócio Sustentável e Inteligência Artificial).

## Objetivo da Página
Esta plataforma web visa apresentar de forma limpa, futurista e altamente profissional as megatendências mundiais da agricultura e inovação tecnológica projetadas até o ano de 2030 (conforme análises de instituições renomadas como a Embrapa). O projeto busca conscientizar e engajar o usuário frente aos desafios de produção alimentar global aliados aos Objetivos de Desenvolvimento Sustentável (ODS) das Nações Unidas, oferecendo interatividade fluida por meio de um quiz de conhecimentos e um formulário premium de captação.

## Recursos Implementados
1. **Design Futurista Premium**: Layout moderno utilizando a paleta de cores solicitada (Verdes-claros, Roxo neon, Azul elétrico, Vermelho escarlate, Laranja de Princeton e Preto) com efeitos visuais e sombras sutis.
2. **Seções Expansíveis (Accordion)**: Componentes construídos nativamente em JavaScript com transições animadas suaves para detalhamento de megatendências e uso de IA.
3. **Imagens Responsivas**: Espaços reservados (`Foto1.png`, `Foto2.png`, `Foto3.png`) formatados na estrutura correta do layout com efeitos hover estilizados.
4. **Formulário de Inscrição**: Posicionado à direita em telas desktop, responsivo em dispositivos móveis, para captação de dados do leitor.
5. **Quiz de Fixação**: Questionário interativo gerando feedbacks imediatos baseados nas respostas selecionadas.
6. **Acessibilidade Completa**:
   - Controle de zoom de fontes dinâmico (+ / -).
   - Modo Claro e AgroFuturo - O Futuro da Tecnologia no Campo

## TemaEscuro alternáveis.
   - Leitura de voz nativa através da **SpeechSynthesis API** filtrando estritamente os textos da seção principal.

## Instruções de Uso
1. Extraia todos os arquivos do pacote `.zip` em uma mesma pasta local.
2. Certifique-se de manter os 3 arquivos essenciais (`index.html`, `style.css` e `script.js`) no mesmo nível de diretório.
3. Para customizar com suas próprias fotos, adicione os arquivos de imagem com os nomes `Foto1.png`, `Foto2.png` e `Foto3.png` na mesma pasta do projeto (ou altere os valores do atributo `src` nas tags correspondentes do `index.html`).
4. Dê um duplo clique ou abra o arquivo `index.html` em qualquer navegador de internet moderno para navegar pela plataforma.
